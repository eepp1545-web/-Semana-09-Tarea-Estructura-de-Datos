Este proyecto consiste en el desarrollo de un sistema de gestión de inventario simple para una tienda, implementado en Python utilizando los principios de la Programación Orientada a Objetos (POO).
El sistema permite:
- Añadir productos
- Eliminar productos
- Actualizar productos
- Buscar productos por nombre
- Listar todos los productos registrados
